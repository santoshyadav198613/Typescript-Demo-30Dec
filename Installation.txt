Typescript: 

1. created by microsoft in 2011, it is open source.
2. It is superset of java.
3. Converts the typescript code to optimized JS code.
4. End result is always a JS code.
5. Transpiler - Tramsforms TS to JS and JS to optimized code.
6. Can be used apart from angular.
